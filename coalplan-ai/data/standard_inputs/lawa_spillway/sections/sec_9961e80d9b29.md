(1)在施工前56d向监理人递交止水材料生产厂家及其合格证书、产品说明及样品，以及安装或埋设止水片的施工措施设计，并在竣工后递交实施的竣工图。

(2)止水设施的型式、尺寸、埋设位置和材料的品种规格符合本工程施工图纸的规定。本工程结构缝和施工缝一般采用铜止水、遇水膨胀橡胶止水条。

(3)紫铜止水片表面光滑平整，并有光泽。其化学成分符合GB/T2059-2017《铜及铜合金带材》的规定，型式与尺寸符合设计文件和规范的有关要求。铜止水片的异型接头在工厂整体冲压成型。铜止水物理力学指标见表3.6-6。

表3.6-6 铜止水物理力学指标表

| 项目 | 单位 | 指标 |
| --- | --- | --- |
| 抗拉强度 | MPa | 不小于205 |
| 延伸率 | % | 大于等于30 |
| 冷弯 |  | 冷弯180°，不出现裂缝，在0°~60°范围内连续张闭50次不出现裂缝 |
| 比重 | g/cm3 | 8.89 |

(4)金属止水片平整、干净、无砂眼和钉孔，采用搭接方式埋设，其搭接长度不小于20mm，搭接部位应采用双面焊接，并保证接合部位密实。

(5)施工时尽量减少止水铜片的接头。采用铜卷材现场压制成形。铜止水片连接采用搭接焊接或对缝焊接。搭接焊接搭接长度大于20mm，要求双面焊接，采用黄铜焊条气焊；若因条件所限不能进行双面焊接时，则采用钨极氩弧焊的单面焊，单面焊要求采用双层焊道的方法进行。对缝焊接采用单面双层焊道焊缝，在对缝焊接后，利用相同止水带形状且宽度不小于60mm的贴片，对称焊接在接缝两侧的止水带上。焊接保证质量，不采用手工电弧焊接，焊接时对垫片作防火、防融蚀保护。焊接接头表面平整光滑、无孔洞和缝隙、不渗水。

(6)止水铜片安装后，采取措施固定牢固，使位置符合设计要求，其误差不超过允许偏差，安装就位后，铜止水片鼻子外缘应涂刷一薄层沥青漆。

(7)橡胶止水带型式、尺寸符合设计文件和规范的有关要求，其拉伸试验按国家标准进行。橡胶止水带物理力学性能满足止水带材料符合规范DL/T5215-2005表5.2.2-1及表5.2.4中有关规定。橡胶止水带接头采用硫化连接，施工前先将两端削成台阶状。接头内不得有气泡或夹渣，中心部分应粘接紧密、连续，拼接处的抗拉强度不小于母材抗拉强度的60%。

(8)橡胶止水条的安装应防止变形和撕裂。橡胶止水粘结不采用再生胶。

(9)安装好的止水条加以固定和采取可靠的保护措施。
