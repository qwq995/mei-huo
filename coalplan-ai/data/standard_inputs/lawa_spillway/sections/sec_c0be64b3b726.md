安全标识标志牌配置见表6.9-1。

表6.9-1 安全标识标志牌配置表

| 序号 | 名 称 | 安全标志<br>类别 | 配置区域及要求 |
| --- | --- | --- | --- |
| 1 | 户外广告牌 | 宣传类 | 户外广告牌应纳入项目法人安全文明施工规划，配置在项目现场醒目位置。 |
| 2 | 施工单位简介 | 宣传类 | 施工单位简介应设置在施工单位办公区门口及施工现场主要部位如：连接道路进口处等。 |
| 3 | 四牌二图 | 宣传类 | “四牌二图”布置应纳入施工组织设计或总平面管理范围，施工单位“四牌二图”应设置在办公区门口位置及施工现场各主要入口处。 |
| 4 | 户外安全宣传栏 | 宣传类 | 户外安全宣传栏分为带雨蓬和不雨蓬两类，带雨蓬宣传栏主要设置在项目部办公区、宿舍区。不带雨蓬宣传栏主要设置在施工现场各主要工作面。 |
| 5 | 安全警告宣传牌 | 安全类 | 安全警告宣传牌主要设置在项目部生活区及施工区各个施工作业面 |
| 6 | 组合安全警示牌 | 安全类 | 组合安全警告牌主要悬挂在施工区各主要施工部位。 |
| 7 | 安全提示板 | 安全类 | 安全提示板应设置在各个施工作业面，内容应包括危险点、危险源提示及应对措施等。 |
| 8 | 安全施工记录牌 | 安全类 | 安全施工记录牌内容包括各施工单位安全施工记录天数。主要设置在项目部办公区、主要施工面。 |
| 9 | 安全文明施工责任区牌 | 安全类 | 安全文明施工责任区牌应包括施工区名称，施工负责人、技术负责人、质量负责人、安全负责人名字及电话。主要布置在各个施工作业面上。 |
| 10 | 操作规程牌 | 安全类 | 操作规程牌主要悬挂各种机械设备上方。 |
| 11 | 安全责任制牌 | 安全类 | 安全责任制牌主要悬挂在施工电源箱柜及各类大型机械设备上。 |
| 12 | 材料标识牌 | 安全类 | 材料标识牌主要放置在各种材料及成品、半成品旁，内容包括材料名称、规格型号、检验状态。 |
| 13 | 脚手架搭设标牌 | 安全类 | 脚手架搭设标识牌主要内容包括搭设人、承重荷载、验收人、验收时间、使用单位、责任人等信息。脚手架通过验收后，悬挂在脚手架上。 |
| 14 | 配电箱责任牌 | 安全类 | 配电箱责任牌内容包括责任单位及责任人。主要悬挂在配电箱上方。 |
| 15 | 禁止烟火 | 禁止类 | 悬挂在透平油罐区域；悬挂在油库、火工品仓库大门旁和有人出入通道口旁以及四周围墙外壁上；悬挂在蓄电池室门上；悬挂在储存易燃、易爆物品仓库门口；悬挂在木工房、油漆场所、油处理室、汽车库内及汽车修理场所；悬挂在控制室、保护仪表盘室等门口和室内；悬挂在电缆夹层、电缆隧道、竖井等入口处； 悬挂在主机房、计算机房、档案室入口处。 |
| 16 | 禁止带火种 | 禁止类 | 悬挂在储存易燃易爆物品仓库入口处，标志牌底边距地面高1.5m左右。 |
| 17 | 禁止攀登<br>高压危险 | 禁止类 | 悬挂在户外高压配电装置构架的爬梯上；悬挂在主变压器、高压备用变压器、高压厂用变压器和电坑器等设备的爬梯上；悬挂在架空电力线路杆塔的爬梯上和配电变压器的杆架或台架上；标志牌的底边距地面高1.5—3.0m。 |
| 18 | 禁止合闸<br>有人工作 | 禁止类 | 悬挂在一经合闸即可送电到已停电检修(施工)设备断路器和隔离开关的操作把手上，悬挂在控制室内已停电检修设备的电源关或合闸按钮上；悬挂在控制屏上的控制牌可根据实际需要制作，可以只有文字，没有图形。 |
| 19 | 禁止合闸<br>线路有人工作 | 禁止类 | 悬挂在已停电检修的(施工)电力线路的断路器或隔离开关操作把手上。 |
| 20 | 禁止操作<br>有人工作 | 禁止类 | 悬挂在已停运检修隔离管道系统的阀门操作手轮上；悬挂在已停运检修隔离烟风系统的档板手柄上。 |
| 21 | 禁止跨越 | 禁止类 | 悬挂在输送皮带、缆绳、热力管道、深坑等到危险场所，面向行人。 |
| 22 | 禁止乘人 | 禁止类 | 悬挂在专用载货的升降吊笼、送料小车、升降机上和入口门旁。 |
| 23 | 禁止吸烟 | 禁止类 | 除有禁止烟火标志外，悬挂在火灾危险性类别在丙级以上的建筑物上，如变压器室、控制室、继电保护室、自动和远动装置室，以及其他禁止吸烟的地方。 |
| 24 | 禁止戴手套 | 禁止类 | 悬挂在车床、钻床、铣床、磨床等所有施转机床和所有旋转机械的作业场所。 |
| 25 | 闲人禁止入内 | 禁止类 | 悬挂在炸药库的门上。 |
| 26 | 禁止游泳 | 禁止类 | 悬挂在禁止游泳的水库、消防池出入口处。 |
| 27 | 施工现场<br>禁止通行 | 禁止类 | 悬挂或放置在禁止通行的检修现场或围栏旁、入口处或可能发生意外伤害的施工现场。 |
| 28 | 止步<br>高压危险 | 警告类 | 悬挂在室外带电设备工作地点的安全围栏上；悬挂在因高压危险禁止通行的过道上；悬挂在室外带电设备构架上；悬挂在工作地点临近带电设备的横梁上；悬挂在室外带电设备固定围栏上。 |
| 29 | 当心触电 | 警告类 | 悬挂在临时电源配电箱上；悬挂在生产现场可能发生触电危险的电器设备和线路上，如：配电室、断电器上等。 |
| 30 | 当心坑洞 | 警告类 | 悬挂在生产现场和通道临时开启或挖掘孔洞四周的围栏上。 |
| 31 | 当心坠落 | 警告类 | 悬挂在高处作业搭设的脚手架栏杆上；悬挂在易发生坠落事故的作业地点。 |
| 32 | 当心落物 | 警告类 | 悬挂在易发生落物的地点，如高处作业的下方、地质条件不稳定的洞室等。 |
| 33 | 必须戴安全帽 | 指令类 | 悬挂在生产场所主要通道入口处。 |
| 34 | 必须系安全带 | 指令类 | 悬挂在高差1.5-2m周围没有设置防护围栏的作业地点；悬挂在高处作业场所 |
| 35 | 必须戴防护眼镜 | 指令类 | 悬挂在车床、钻床、砂轮机等；悬挂在焊接和金属切割工作场所；悬挂在化学处理、使用腐蚀或其他有害物品的场所。 |
| 36 | 必须戴安全帽 | 指令类 | 悬挂在机加工车间入口处。 |
| 37 | 配戴防护手套 | 指令类 | 悬挂在易伤害手部的作业场所，如：具有腐蚀、污染、灼烫等危险的作业地点。 |
| 38 | 从此上下 | 提示类 | 悬挂在现场工作人员可以上下的通道、爬梯上。 |
| 39 | 消火栓箱 | 消 防<br>及其他 | 消火栓前面部示范：标注消火栓、火警电话和厂内电话号码及编号等字样；结构：四周为木质或其他金属材料制作，面部为玻璃材料；应急工具：侧面应悬挂一粗钢筋制成的小锤。 |
| 40 | 灭火器 | 消 防<br>及其他 | 悬挂在灭火器、灭火器箱的上方；悬挂在灭火器、灭火器箱存放的通道上；泡沫灭火器身上应标志“不适应电火”字样。 |
| 41 | 灭火器箱 | 消 防<br>及其他 | 灭火器箱前面部示范：灭火器箱、火警电话、<br>厂内电话号码、编号等字样；泡沫灭火器箱上应在其顶面部标志“不适应电火”字样；结构：金属框架四面采用玻璃或金属材料。 |
| 42 | 急救药箱 | 消 防<br>及其他 | 张贴在现场使用的急救药箱上。 |
| 以上各类安全标志配置依据：GB2894-2008《安全标志及其使用导则》、DL26860—2011《电业安全工作规程》、GB13495-92《消防安全标志》、DL5027—93《电力设备典型消防规程》、GB6389-1997《工业企业铁路道口安全标准》 |  |  |  |
